# Lecture Review Project

Frontend : Vue.js 3
역할 : 사용자 UI 구현, 강의 목록 후기 작성 등 인터페이스 담당
선정이유 : Vue.js는 컴포넌트 기반 구조로 UI를 효율적으로 개발할 수 있으며 ,학습 난이도가 비교적 낮아 팀원들이 빠르게 적응할 수 있다 또한 API 연동이 용이하여 백엔드와의 통신에 적합하다.

Backend
FastAPI: API 서버 구축 — 빠른 개발 속도와 API 문서 자동 생성
SQLModel: 데이터 모델링 — DB 설계와 데이터 검증의 단일화
JWT: 사용자 인증 — 상태 유지가 필요 없는 효율적인 토큰 인증
bcrypt: 데이터 보안 — 사용자 비밀번호 단방향 암호화 보안

Database
Neon (PostgreSQL): 데이터 저장소 — 클라우드 기반의 안정적인 협업용 DB

DevOps / Tools
uv: 패키지 관리 — 초고속 라이브러리 관리 및 가상 환경 구축
Swagger UI: 협업 도구 — 프론트엔드 공유용 실시간 API 테스트 문서
